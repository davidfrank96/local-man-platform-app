# Localman PWA Icons

Generated from the official PNG source asset for Phase 0 PWA groundwork.
These icons use a white background. Phase 1 service-worker support may cache
these static icon files only; it must not cache dynamic marketplace data.

Current icon set:
- Android/PWA: `72`, `96`, `128`, `144`, `152`, `192`, `256`, `384`, and `512` pixels
- Apple/iOS: `120`, `152`, `167`, and `180` pixels, plus `apple-touch-icon.png`
- Browser: `16`, `32`, and `48` pixel favicons
- Maskable: `192` and `512` pixels
